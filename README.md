# memo-app-php
PHPとMySQLを用いて作成した、ログイン機能付きメモ管理アプリです。
※ 本アプリのソースコードは master ブランチに格納しています。

## 概要
ユーザー登録・ログイン機能を備えた、個人用メモ管理Webアプリです。  
ログイン中のユーザーごとにメモを管理できるよう設計しています。

## 主な機能
- ユーザー登録機能
- ログイン / ログアウト機能（セッション管理）
- メモの作成・一覧表示
- ユーザーごとのデータ管理

## 使用技術
- PHP（プレーンPHP）
- MySQL
- HTML / CSS

## 工夫した点
- パスワードをハッシュ化して保存し、セキュリティを意識した実装を行いました
- セッションを利用してログイン状態を管理しています
- データベース接続情報を `db.sample.php` として分離し、公開時の安全性を考慮しています

## 環境構築について
1. MySQLでデータベースを作成
2. `db.sample.php` をコピーして `db.php` を作成
3. データベース接続情報を設定
4. ブラウザで `index.php` にアクセス

※ 本リポジトリではセキュリティの観点から `db.php` は公開していません。

## 補足
本アプリは、PHPとデータベース連携の基礎理解を目的として作成しました。
